# Dependencies
1. Python 3 (for UI client)
2. [tkinter](https://docs.python.org/3/library/tkinter.html) (for UI client)
3. [Parse](https://pypi.org/project/parse/#description) (for UI client)
3. gcc (for command line client and server)
